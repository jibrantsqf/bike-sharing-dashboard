# 🚲 Bike Sharing Data Analysis Dashboard

## 📌 Deskripsi Proyek
Proyek ini adalah tugas akhir dari kelas **Belajar Analisis Data dengan Python** di Dicoding. Tujuan dari proyek ini adalah untuk menganalisis dataset *Bike Sharing* dan menggali *insight* mengenai pengaruh kondisi cuaca, musim, serta pola perilaku penyewaan sepeda berdasarkan jam sibuk (pelanggan *Casual* vs *Registered*).

## 📊 Fitur Dashboard
Dashboard interaktif dibuat menggunakan **Streamlit** dengan fitur:
- Filter rentang waktu (Kalender) yang dinamis.
- Ringkasan metrik total penyewaan.
- Visualisasi pengaruh musim dan cuaca terhadap penyewaan.
- Analisis pola jam sibuk (*clustering* waktu).
